# 内容抽取与结构化策略

## 目标

从教材 PDF 提取关键知识，重写为紧凑速成讲义。覆盖完整 + 快速复习，不是逐字翻译。

## 密度控制

- 每章输出控制在原教材页数的 1/4 到 1/3
- 优先保留：定义、定理、核心公式、经典例题
- 可精简：冗长证明、重复例题、历史背景
- 每章保留 2-4 道最典型例题

## 章节结构模板

每章按以下结构组织：

```
<h2 class="chapter" id="chN">第N章 标题</h2>

<div class="box tip"><span class="label">本章导读</span>
[2-3句话概述 + 考试地位 + 重点提示]
</div>

<h3 class="section" id="chN-M">N.M 小节标题</h3>
[定义->定理->公式->例题->警示->技巧，按需穿插]

<div class="box method"><span class="label">本章解题流程</span>
[1-2-3步骤总结]
</div>
```

## 公式书写规范

- 行内公式：`$E=mc^2$`
- 独立公式：`$$P(A)=k/n$$`
- 所有数学必须用 LaTeX，不要用 Unicode 数学符号
- 多行公式和矩阵直接用 LaTeX 环境包裹

## 表格规范

- 使用 `<table class="tbl">`（三线表：顶线+表头线+底线，无单元格边框）
- `<thead>` 包表头，`<tbody>` 包数据
- 分布表、速查表统一放附录区

## 图片处理

详见 `image-strategy.md`。原则：**重绘优先于提取**。

## 附录区内容

正文后自动生成：
1. 公式速查总表（按章汇总所有 box.formula）
2. 易错点总汇（按章汇总所有 box.warn）
3. 关键数据表（常用分布表、分位数表等）
4. 考试策略（时间分配 + 检查清单）
5. 速成时间表（如果用户提供了考试日期）

## 语言

保持教材原文语言。中文教材输出中文，英文教材用中文翻译并保留关键英文术语。
